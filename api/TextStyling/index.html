TextStyling

This class was designed to simplify the colours used in text and allows you to use colour & format codes which then are converted into font tags for rich text.

Colour Codes

&0 - Black
&1 - Dark Blue
&2 - Dark Green
&3 - Dark Aqua
&4 - Dark Red
&5 - Dark Purple
&6 - Orange
&7 - Gray
&8 - Dark Gray
&9 - Blue
&a - Green
&b - Aqua
&c - Red
&d - Light Purple
&e - Yellow
&f - White
&g - Gold

Format Codes

&l - Bold
&m - Strikethrough
&n - Underline
&s - Stroke
&o - Italic
&r - Reset
